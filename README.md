# Обзор платформы Remote.Team

Этот репозиторий содержит статью-обзор сервиса [Remote.Team](https://remote.team) — защищённой веб-платформы для командной работы, объединяющей чат и управление задачами.

Статья опубликована в виде статического сайта через GitHub Pages и доступна по адресу:  
[https://github.com/sdmitruk-creator/remote-team-taskmanager](https://github.com/sdmitruk-creator/remote-team-taskmanager)

## Что внутри

- `index.md` — полный текст обзора с Markdown-разметкой, оптимизированный для GitHub Pages.
- `README.md` — вы сейчас здесь.

## Зачем это нужно

Цель репозитория — предоставить читаемый, хорошо структурированный обзор Remote.Team с честным сравнением с Basecamp, Битрикс24 и Weeek. Материал можно использовать как справочник, делиться ссылкой с командой или предложить улучшения через pull request.

## Как помочь

Нашли неточность, опечатку или хотите добавить актуальную информацию?  
Буду рад вашим issue и pull request’ам. Давайте делать обзор ещё лучше вместе.
